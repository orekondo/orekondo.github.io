AWSについて★★★

Posted at — Mar 13, 2020

Amazon Web Services

日本には5番目のリージョンとして東京に2011年からスタート
もともとはAmazonの自社基盤

クラウドの中身

アベイラビリティゾーン（AZ）
→近くのデータセンターの集まり

マルチAZ配置
→高可用性を目的とした冗長パターン
　AZは世界で64個くらいある
　日本には東京と大阪（大阪はバックアップ用？）
　リージョン＝AZを地域でまとめたもの（世界で21個くらいある）

エッジロケーションサービス（グローバスサービス）
→CDN,DNS,セキュリティとかリージョンに委譲しないサービスが入ってる

操作方法

AWSマネジメントコンソール
→WEBベースのユーザーインターフェース

AWS Command Line Interface(CLI)
→いわゆる黒い画面で操作するやつ

AWS SDK
→開発キット Javaとか.NetとかRubyとかPythonとかnode.jsとかで開発したアプリに組み込むやつ
