Hugoのめも

Posted at — Mar 7, 2020

Hugoって

静的サイトジェネレータ

htmlをあらかじめ作っておくからDBとかいらない
→静的だからハッキングとか関係ないから安心安全

DBとかいらないってことはWebサーバだけうごけばどこでもいい
→単純にCDNとかかますだけでつよつよになる

画像の管理とかは自分でやらないといけないからちょっとめんどくさい
ハンドメイド感がたのしい

Markdown記法で書く

記号で文章に装飾できるやつ
ある程度フォーマットが縛られるから賢い風の見た目になりがち

コマンドめも

新しいサイトを作る

hugo new site <サイト名>

新しい記事をつくる

hugo new posts/<ファイル名>.md

ローカルでサーバを立ち上げる

# draftの記事は表示されない
hugo server
# draftの記事も表示される
hugo server -D
# ファイルの更新を検知して再構成してくれる
hugo server -w

アップロードするファイルを生成する

hugo
